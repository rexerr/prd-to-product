# Active decisions

Thin index of currently-binding decisions whose constraints aren't obvious from reading code alone — **one line per decision, the binding rule only**. IDs link the full append-only entry in [`DECISIONS.md`](DECISIONS.md); read it there for rationale, context, and verification. Implementation detail that's visible by reading the code does **not** belong here (that's what made this file accrete).

A decision belongs here if **all** are true: it imposes a rule the agent must follow now; the rule is not enforced or visible by reading the code itself; it has not been superseded. When a new decision meets the bar, mirror it as a one-liner; when one is superseded, the `/end-session` retirement ritual drops it.
